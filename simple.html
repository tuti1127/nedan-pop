このツールについて

値段POP作成ツールは、園芸店のスタッフが手書きPOPを作る手間を減らすために開発した業務ツールです。 商品名と税抜価格を入力するだけで、A4用紙に印刷できるきれいな値段POPを自動生成します。

主な利用者は店頭スタッフ。スマホ・タブレット・PCのどれでも動き、 インターネット接続もサーバーも不要。HTMLファイルをダブルクリックするだけで使えます。

A4に出力

ステップ1〜2は何度でも繰り返せます。複数の商品を追加してから、まとめて1枚の紙に印刷することが可能です。 リアルタイムプレビューで、追加前に仕上がりイメージを確認できます。

技術的な特徴
単一HTMLファイル
CSS・JavaScript・HTMLがすべて1ファイルに収まっています。サーバー不要、インターネット不要。ダブルクリックで起動します。
外部依存なし
jQueryもReactも使いません。素のHTML/CSS/JSだけで構成しています。将来にわたって動き続けます。
A4印刷最適化
@media print で印刷時のみ専用レイアウトを適用。ブラウザの印刷機能をそのまま使えます。
レスポンシブ対応
スマホ（320px〜）からPCまで対応。スタッフが店頭でスマホを使っても問題なく操作できます。
CSS変数設計
色・余白・角丸をCSS変数で一元管理。デザイン変更は変数を1か所書き換えるだけで全体に反映されます。
GitHub Pages配信
GitHubの無料ホスティング機能を使ってURLから誰でもアクセス可能。更新もgit pushで自動反映されます。

ファイル構成

このツールは simple.html 1ファイルだけで構成されています。 HTMLファイルの中にCSS・JavaScriptが埋め込まれており、外部ファイルへの依存は一切ありません。

simple.html ← これ1つだけ！ │ ├─ <style> ← CSS部分（見た目のスタイル） │ ├─ :root { --変数 } ← デザイントークン │ ├─ body, header, .tab-bar ← レイアウト │ ├─ フォーム部品 ← input, button など │ ├─ .pop-card ← POPカードの見た目 │ └─ @media print ← 印刷専用スタイル │ ├─ <body> ← HTML部分（画面の構造） │ ├─ .header ← タイトルバー │ ├─ .tab-bar ← タブボタン3つ │ ├─ #tab-create ← POP作成タブ │ ├─ #tab-design ← 設計まとめタブ │ ├─ #tab-code ← コード解説タブ │ └─ #print-area ← 印刷専用の非表示エリア │ └─ <script> ← JavaScript部分（動きのロジック） ├─ switchTab() ← タブ切替 ├─ addItem() ← 商品追加 ├─ renderList() ← リスト描画 ├─ renderPreview() ← A4プレビュー描画 └─ printPops() ← 印刷実行
コードの読み方

このページでは、ツールを構成するコードを CSS → HTML → JavaScript の順に紹介します。 各コードブロックの前に「ここは何をしているか」の解説を入れていますので、 プログラミング未経験の方でも雰囲気をつかめるように心がけました。

コード中の /* こんな書き方 */ がコメント（説明文）です。 実際の動きには影響せず、読む人のための注釈です。

CSS 見た目のスタイル
CSS変数（デザイントークン）とは
CSSの :root { --変数名: 値; } という書き方で、 色や余白などをまとめて定義できます。あとで「この緑を少し濃くしたい」となったとき、 この1か所を変えるだけで全体が一括更新されます。
タブバーのスタイル
3つのタブボタンを横並びにして均等幅にします。 アクティブなタブだけ下線（border-bottom）が緑になります。
印刷専用スタイル（@media print）
@media print の中に書いたスタイルは、 「印刷するとき（またはPDF出力のとき）だけ」適用されます。 タブバーやヘッダーを消して、POPカードだけをA4いっぱいに表示します。
HTML 画面の構造
タブバーのHTML
ボタンを3つ並べて、クリック時に switchTab() 関数を呼び出します。 どのタブを表示するか（tab-create / tab-design / tab-code）を引数として渡しています。
商品追加フォームのHTML
商品名・価格の入力欄と、税率選択ボタン・追加ボタンで構成されます。 id="name-input" などのIDを使って、 JavaScriptから値を取得・操作します。
JS 動きのロジック
タブ切替（switchTab）
ボタンをクリックすると、全タブを非表示にしてから、クリックされたタブだけを表示します。 CSS の display: none / block を JavaScriptから切り替えるシンプルな仕組みです。
商品追加（addItem）
入力値を検証して、問題なければ items 配列に追加します。 バリデーション（入力チェック）に失敗したときは、エラー状態のCSSクラスを付けて赤い枠線を表示します。
A4プレビュー描画（renderPreview）と印刷（printPops）
グリッドレイアウト（CSS Grid）を使って、選択した面数（1〜16）に合わせてPOPカードを配置します。 印刷ボタンを押すと非表示エリアに同じカードを生成してから window.print() を呼び出します。
リアルタイムプレビュー（updateLivePreview）
入力欄の input イベントを監視して、 キーを押すたびにプレビューカードを更新します。まだ何も入力していないときは薄いグレーのプレースホルダー表示にします。
